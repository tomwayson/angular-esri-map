Description

Functions to help create SceneView instances.

Dependencies

Methods

  • getSceneView()

    Load and get a reference to a SceneView module

    Returns

    Returns a $q style promise which is resolved with an object with a view property that refers to the SceneView

  • setMap(map)

    Set a Map or WebScene on the SceneView. A new SceneView will be constructed if it does not already exist, and also execute the optional on-load and on-create events. If a new SceneView is rejected, the optional on-error event will be executed.

    Parameters
    ParamTypeDetails
    mapObject

    Map instance or WebScene instance

    Returns
    Promise

    Returns a $q style promise and then sets the map property and other options on the SceneView.
