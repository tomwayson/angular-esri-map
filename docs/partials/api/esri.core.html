esri.core Module

The esri.core module includes services used by the directives in the esri.map module. These services can also be used directly by your own custom directives.

Loading Esri Modules

Use esriLoader to lazy load the Esri ArcGIS API or to load API modules.

Getting a Reference to a MapView or SceneView

Use the esriRegistry to store and retrieve Esri MapView or SceneView instances for use in different controllers.
