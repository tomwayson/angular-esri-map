esriMap
directive in module esri.map

Description

This is the directive which will create a map using the Esri ArcGIS API for JavaScript. There are plenty of examples showing how to use this directive and its bound parameters. For additional information, we recommend the Esri Map documentation.

Examples

  • Choose from a number of examples making use of this directive

Usage

as element:
<esri-map
       id="{String}"
       [center="{Array | Object}"]
       [zoom="{Number}"]
       [item-info="{Object}"]
       [basemap="{String}"]
       [load="{Function}"]
       [extent-change="{Function}"]
       [map-options="{Object | String}"]
       [register-as="{String}"]>
</esri-map>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
idString

The id of the element where to construct the map.

center
(optional)
Array Object

The initial location of the map.

zoom
(optional)
Number

The initial zoom level of the map.

item-info
(optional)
Object

When loading a web map from an item id, this object will be populated with the item's info.

basemap
(optional)
String

The basemap of the map, which can be a valid string from the ArcGIS API for JavaScript, or a custom basemap.

load
(optional)
Function

Callback for map load event.

extent-change
(optional)
Function

Callback for map extent-change event.

map-options
(optional)
Object String

An object or inline object hash string defining additional map constructor options. See Additional Map Options example.

register-as
(optional)
String

A name to use when registering the map so that it can be used by parent controllers. See esriRegistry.
