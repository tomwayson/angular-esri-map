EsriWebsceneSlidesController
controller in module esri.map

Description

Functions to help create and manage individual slide properties and behaviors.

Methods

  • onSlideClick(slide)

    This method is executed when an individual slide is clicked. It passes the slide object to the on-slide-change bound callback function, which can be useful for manipulating an associated Esri Scene View. It also sets the clicked slide's active status to true, to assist with CSS styling.

    Parameters
    ParamTypeDetails
    slideObject

    slide instance

  • setSlides(slides)

    Set a an array of slides for the directive. Each slide will have its own entry in the template.

    Parameters
    ParamTypeDetails
    slidesArray

    Array of Slide instances.
