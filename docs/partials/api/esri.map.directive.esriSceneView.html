esriSceneView
directive in module esri.map

Description

This is the directive which will create a scene view using the Esri ArcGIS API for JavaScript. There are plenty of examples showing how to use this directive and its bound parameters.

Examples

  • Scene View
  • Extrude Polygon
  • Toggle Basemap Elevation
  • and more...

Usage

as element:
<esri-scene-view
       map="{Object}"
       [on-create="{Function}"]
       [on-load="{Function}"]
       [on-error="{Function}"]
       [view-options="{Object | String}"]
       [register-as="{String}"]>
</esri-scene-view>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
mapObject

Instance of a Map or WebScene.

on-create
(optional)
Function

Callback for successful creation of the scene view.

on-load
(optional)
Function

Callback for successful loading of the scene view.

on-error
(optional)
Function

Callback for rejected/failed loading of the scene view, for example when WebGL is not supported.

view-options
(optional)
Object String

An object or inline object hash string defining additional scene view constructor options.

register-as
(optional)
String

A name to use when registering the view so that it can be used by other controllers. See esriRegistry.
