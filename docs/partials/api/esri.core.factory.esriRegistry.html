esriRegistry
service in module esri.core

Description

Use esriRegistry to store and retrieve MapView or SceneView instances for use in different controllers.

Examples

  • Registry Pattern

Methods

  • get(name)

    Get the MapView or SceneView instance registered with the given name. See esriMapView or esriSceneView for info on how to register a map using the register-as attribute.

    Parameters
    ParamTypeDetails
    nameString

    Name that the view was registered with.

    Returns
    Promise

    Returns a $q style promise which is resolved with the view once it has been loaded.
