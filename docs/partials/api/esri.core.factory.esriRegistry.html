esriRegistry
service in module esri.core

Description

Use esriRegistry to store and retrieve map instances for use in parent controllers.

Examples

  • Registry Pattern

Methods

  • get(name)

    Get the map instance registered with the given name. See esriMap for info on how to register a map using the resgister-as attribute.

    Parameters
    ParamTypeDetails
    nameString

    Name that the map was registered with.

    Returns
    Promise

    Returns a $q style promise which is resolved with the map once it has been loaded.
