esriFeatureLayer
directive in module esri.map

Description

This directive creates a FeatureLayer and adds it to the map. This directive must be placed within an esri-map directive.

Examples

  • Feature Layers
  • Add/Remove Layers
  • Layer Events
  • No Basemap
  • and more...

Usage

as element:
<esri-feature-layer
       url="{String}"
       [visible="{Boolean}"]
       [opacity="{Number}"]
       [definition-expression="{String}"]
       [load="{Function}"]
       [update-end="{Function}"]
       [layer-options="{Object | String}"]>
</esri-feature-layer>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
urlString

The url to the ArcGIS Server REST layer resource.

visible
(optional)
Boolean

The visibility of the layer. Two-way bound.

opacity
(optional)
Number

The opacity of the layer. Two-way bound.

definition-expression
(optional)
String

The definition expression where clause. Two-way bound.

load
(optional)
Function

Callback for layer load event.

update-end
(optional)
Function

Callback for layer update-end event.

layer-options
(optional)
Object String

An object or inline object hash string defining additional layer constructor options.
