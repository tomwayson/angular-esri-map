esriHomeButton
directive in module esri.map

Description

This is the directive which will create a home button using the Esri ArcGIS API for JavaScript.

Examples

  • Home Button

Usage

as element:
<esri-home-button
       view="{Object}">
</esri-home-button>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
viewObject

Instance of a MapView or SceneView.
