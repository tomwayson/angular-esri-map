esriLegend
directive in module esri.map

Description

Show a legend for the map in the specified element on the page. The legend directive must be placed within an esri-map directive, but the target element (that will contain the legend) can be anywhere on the page.

Examples

  • Legend
  • Web Map
  • Additional Map Options

Usage

as element:
<esri-legend
       target-id="{string}">
</esri-legend>
as attribute
<ANY esri-legend
     target-id="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
target-idstring

The id of the element where you want to place the legend
