esriVectorTileLayer
directive in module esri.map

Description

This directive creates a VectorTileLayer and adds it to the map. This directive must be placed within an esri-map directive.

Examples

  • Vector Tile Layer

Usage

as element:
<esri-vector-tile-layer
       url="{String}"
       [visible="{Boolean}"]
       [opacity="{Number}"]
       [load="{Function}"]
       [layer-options="{Object | String}"]>
</esri-vector-tile-layer>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
urlString

The url to the ArcGIS Server REST layer resource.

visible
(optional)
Boolean

The visibility of the layer. Two-way bound.

opacity
(optional)
Number

The opacity of the layer. Two-way bound.

load
(optional)
Function

Callback for layer load event.

layer-options
(optional)
Object String

An object or inline object hash string defining additional layer constructor options.
