esriDynamicMapServiceLayer
directive in module esri.map

Description

This directive creates an ArcGISDynamicMapServiceLayer and adds it to the map. This directive must be placed within an esri-map directive.

Examples

  • Dynamic Map Service Layer
  • Multiple Dynamic Map Service Layers
  • Layer Events

Usage

as element:
<esri-dynamic-map-service-layer
       url="{String}"
       [visible="{Boolean}"]
       [opacity="{Number}"]
       [visible-layers="{Array|String}"]
       [load="{Function}"]
       [update-end="{Function}"]
       [layer-options="{Object | Function}"]>
</esri-dynamic-map-service-layer>

Directive info

  • This directive creates new scope.

Parameters

ParamTypeDetails
urlString

The url to the ArcGIS Server REST map service resource.

visible
(optional)
Boolean

The visibility of the layer. Two-way bound.

opacity
(optional)
Number

The opacity of the layer. Two-way bound.

visible-layers
(optional)
ArrayString

The visible sublayer ids of the map service, either an array of integers or a string of comma-separated integers. Two-way bound.

load
(optional)
Function

Callback for layer load event.

update-end
(optional)
Function

Callback for layer update-end event.

layer-options
(optional)
Object Function

An object or inline object hash string defining additional layer constructor options.
