EsriFeatureLayerController
controller in module esri.map

Description

This controller is used by the esri-feature-layer directive to construct the layer, provide it with several supporting methods, and help bind layer events and other properties such as layer visibility.

Dependencies

  • $controller
  • esriLayerUtils
  • EsriLayerControllerBase
